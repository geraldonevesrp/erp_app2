# Estrutura do Banco de Dados Supabase

Query utilizada para extrair a estrutura:
```sql
SELECT

  table_name,
  column_name,
  data_type,
  is_nullable,
  column_default
FROM

  information_schema.columns
WHERE

  table_schema 
=
 
'public'

ORDER
 
BY

  table_name,
  ordinal_position;
```
